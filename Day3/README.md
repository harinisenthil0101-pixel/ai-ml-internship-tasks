 Project Overview

This project focuses on predicting house prices using **Linear Regression**, a fundamental machine learning algorithm. The goal is to understand how different features of a house (such as area, number of bedrooms, etc.) affect its price.

---

 Objective

* Implement **Simple & Multiple Linear Regression**
* Understand regression modeling
* Evaluate model performance using standard metrics

--
Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

 Dataset

* Housing Price Prediction Dataset
* Contains various features like area, bedrooms, bathrooms, etc.

---

 Steps Involved
 1. Data Preprocessing

* Loaded dataset using Pandas
* Handled categorical variables using encoding
 2. Train-Test Split

* Split dataset into training and testing sets (80:20)
 3. Model Training

* Used Linear Regression from Scikit-learn
* Trained model on training data
 4. Prediction

* Predicted house prices using test data
 5. Model Evaluation

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

---

 Results

* The model was able to predict house prices with reasonable accuracy
* R² score indicates how well the model fits the data

---
 Visualization

* Scatter plot of **Actual vs Predicted Prices**
* Helps in understanding model performance

---

 Key Learnings

* Basics of regression models
* Importance of data preprocessing
* Evaluation of machine learning models

---

 Interview Questions Covered

* Assumptions of Linear Regression
* Interpretation of coefficients
* Difference between MAE and MSE
* Understanding R² score
* Multicollinearity

---

 How to Run the Project

1. Open Google Colab
2. Upload the dataset (`Housing.csv`)
3. Run the notebook step by step
4. View results and graphs

---

 Conclusion

Linear Regression is a simple yet powerful algorithm for predicting continuous values. This project helped in understanding both theoretical and practical aspects of regression.
